# 输出格式模板

> 这是一个占位文件。`SKILL.md` 引用 `references/format.md` 时会加载这里的内容。
> 真实使用时，把这一段引文删掉，替换成你的实际模板。

## 默认模板

```text
标题：<主题>

要点：
- <要点 1>
- <要点 2>
- <要点 3>

行动项：
- [<责任人>] <事项> — 截止 <日期>
```

## 变体 A：简短版

适合 5 分钟以内的快速同步会议。

```text
TL;DR：<一句话总结>

谁负责什么：
- <责任人 1> → <事项 1>
- <责任人 2> → <事项 2>
```

## 变体 B：详细版

适合季度评审、跨部门同步。

```text
# <主题>

## 背景
<2-3 句话>

## 决策
- <决策 1>
- <决策 2>

## 行动项
| 责任人 | 事项 | 截止日期 |
| ------ | ---- | -------- |
| ...    | ...  | ...      |

## 下次同步
<日期 / 议题>
```
