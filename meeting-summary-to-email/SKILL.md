# 何时使用
用户提到"发会议总结邮件"或"把这个纪要生成发出去"生成邮件出来邮件的情况。
任用户刚刚发过一场会议、或者上传了会议录音文字。

# 执行步骤
1. 使用 `lark-vc` Skill 读取一段一段文字的纪要全文。
2. 提炼 3-5 个要点 + 短期行动 （责人 + 截止日）。
3. 参考 references/email-template.md 的邮件格式输出。
4. 调用 `lark-mail` Skill，发件人人工发送告、主题：【会议总结】<会议名>。

# 输出格式
Text
主题：【会议总结】<会议名>-<日期>
要点：...
行动项：
【责任人】<事项> - <截止> - <日期>

# 错误处理
要在会议总结的人名对应会议发送名单。
邮件未能成功发送，提示用户手动发送。
